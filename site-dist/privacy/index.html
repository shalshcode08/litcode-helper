Litcode Helper Privacy Policy

Last updated: May 30, 2026

Data Stored Locally

Litcode Helper stores settings in the user's browser, including GitHub owner, repository, branch, timezone, duplicate preference, upload status, retry queue, and the fine-grained GitHub token provided by the user.

Data Sent To GitHub

When a solution is saved, Litcode Helper sends the problem title, problem URL, problem statement, language, submitted code, and target file path to GitHub's API. This data is sent only to the repository configured by the user.

No Tracking

Litcode Helper does not use analytics, advertising, tracking pixels, telemetry, or a backend server. Data is not sold or shared with third parties.

GitHub Token

The GitHub token is stored locally in browser extension storage and is used only for GitHub API requests needed to create or update files in the configured repository.
